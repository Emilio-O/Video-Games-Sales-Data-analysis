# 🎮 Proyecto Integrativo: Visualización de Datos de Videojuegos

## Descripción

Este proyecto consiste en el desarrollo de una página web interactiva en **Quarto** para explorar y visualizar una base de datos de ventas de videojuegos. El objetivo es aplicar principios de análisis de datos, visualización y control de versiones mediante Git y GitHub.

La página presenta diferentes ejercicios de análisis que incluyen preparación de datos, generación de gráficas e interpretación de resultados.

## Objetivos

* Construir un sitio web utilizando **Quarto**.
* Aplicar técnicas de limpieza y transformación de datos en **R**.
* Crear visualizaciones informativas con **ggplot2**.
* Documentar el proceso de análisis de forma reproducible.
* Utilizar **Git** y **GitHub** para el control de versiones y publicación del proyecto.

## Contenido

El proyecto incluye:

* Descripción general del análisis.
* Exploración inicial de los datos.
* Limpieza y preparación de la base de datos.
* Desarrollo de visualizaciones estadísticas.
* Interpretación de resultados.
* Publicación del sitio web mediante GitHub Pages.

## Herramientas utilizadas

* R
* Quarto
* ggplot2
* dplyr
* Git
* GitHub Pages
